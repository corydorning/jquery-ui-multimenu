Multimenu Plugin

multimenu is a jQuery UI widget that transforms a group of checkboxes/radio buttons and turns them into a dropdown list to provide a solid user expereience and maximize screen real estate.

Options

You can set any of the following options to customize it to fit your needs:


// default options
$('selector').mulitmenu({
  autoOpen: false,
  disabled: false,
  hide: null,
  show: null,

  noneSelectedText: '-- Select Options --',
  selectedList: 0,
  selectedText: '# selected',
  leafNodes: false,
  minButtonWidth: 200,


  header: true,
  selectAll: true,
  selectAllText: 'Select All',
  search: true,
  scrollDelay: 200,

  menuHeight: 200,
  minMenuWidth: 260
});

NOTE: Options can be set using the data-options attribute as well, and passing it an object. For example:


<ul data-role="multimenu" data-options='{ "autoOpen": true, "selectedText": "# of # Employees selected" }'>...</ul>

Events

  • selectall: Triggered when the multimenu's 'Select All' is selected (ex. $(selector).on('selectall', function() { ...do something... });
  • deselectall: Triggered when the multimenu's 'Select All' is deselected (ex. $(selector).on('deselectall', function() { ...do something... });
